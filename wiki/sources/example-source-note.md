# Example Source Note

## Summary

This source argues that a good research repository should preserve both evidence-backed knowledge and idea pressure, without collapsing them into one undifferentiated note stream.

## Key Claims

- Raw inputs should stay immutable.
- Derived source notes should hold summaries, key claims, and thought entries.
- Problem-space pages should carry both knowledge gaps and early `idea_seeds`.
- Over-ingestion lowers signal quality and makes future thinking harder.
