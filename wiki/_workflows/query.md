# Query Workflow

## Purpose

Answer questions against the existing wiki and file durable results back into the vault when useful.

## Steps

1. Start from [index.md](../index.md).
2. Read the smallest set of relevant source, concept, and problem-space pages.
3. Answer with explicit note-level grounding.
4. If the answer introduces durable synthesis, create or update:
   - a concept page, or
   - a problem-space page.
5. Update `Link Reasoning` where a new link is introduced.
6. Append a query result to [log.md](../log.md) if the query materially changed the vault.
