# Recommend Reading Workflow

## Purpose

Given a target problem space, recommend what to read next to improve understanding.

## Steps

1. Read the target problem-space page.
2. Inspect `Knowledge To Acquire Next`, related sources, and related concepts.
3. Explain what is already known.
4. Explain which gaps remain.
5. Recommend reading directions or source types that would reduce those gaps.

## Constraint

- Recommendations are for knowledge acquisition only.
- Do not convert them into execution tasks or delivery checklists.
