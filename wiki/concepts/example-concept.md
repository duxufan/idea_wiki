# Selective Ingestion Preserves Signal

## Core Idea

A useful idea-and-knowledge repository should reject low-information duplicates and unrelated material by default so the graph stays interpretable.

## Supporting Sources

- [[example-source-note]]

## Tension Or Caveat

Over-filtering can hide useful weak signals, so there should still be an explicit path for user-forced ingest when context matters more than immediate relevance.
