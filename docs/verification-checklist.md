# Verification Checklist

- Confirm the minimal public template files exist.
- Confirm source note templates contain `Thought Entries` and `Link Reasoning`.
- Confirm problem-space pages contain `Knowledge To Acquire Next` and no execution checklist.
- Confirm each source note has `source_id` and `raw_path`.
- Confirm the raw registry and sample source note point to the same source.
- Confirm promotion rules distinguish local-only thoughts from promoted ones.
- Confirm `AGENTS.md`, `wiki/_workflows/`, and `docs/commands/` describe ingest, problem-space evolution, and recommend-reading flows.
- Confirm bundled skills reference the repository contract rather than machine-local absolute paths.
